SISTEMA DE EVALUACIÓN DE CONTENIDO DE STREAMING

Este sistema es una aplicación de consola desarrollada para Visual C#, que simula el proceso de evaluación de contenidos de una plataforma streaming 
Permite analizar diferentes características como el tipo de contenido, su duración, su clasificación, el horario en el que será transmitido, y 
el nivel de producción que tiene. Pasando por un proceso de verificación si cumple con las normas técnicas. Dependiendo de los datos dados se 
determinará si el contenido será: Publicado,necesita ajustes, debe enviarse a revisión o si será rechazado.

Instrucciones para usar el sistema:
1) Ejecutar el archivo principal (Program.cs)
2) En el menú que aparece en pantalla, seleccionar una opción ingresando el número correspondiente
3) Si se elige evaluar contenido, ingresar los datos solicitados
4) El sistema mostrará el resultado de la evaluación en la consola

   Explicación:
Este proyecto consiste en el uso de diferentes estructuras de control como condicionales en C#, así como lo son: (if,switch) y ciclos (while,
do while, for) para tomar decisiones de forma automática.
El sistema realiza una validación técnica para verificar que el contenido cumpla con todas las reglas establecidas, después calcula el nivel de
impacto (alto, medio, bajo) y finalmente toma una decisión si que se basa en estos resultados.
Además, el sistema permite mostrar estadísticas, reglas y reiniciar los datos durante su ejecución.
